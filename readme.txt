Poolito Internet Payment Gateway for WooCommerce stores

**Poolito Payment Gateway for WooCommerce**
This plugin enables you to easily set up a payment gateway for WooCommerce payments.
If you need to sell your products and services online, you have to create WooCommerce store and use this plugin to connect to your bank payment gateway.

= Features =
 * Automatically added Currencies to WooCommerce
 * Compatible with WooCommerce 2.6 and older
 * simple and user friendly admin panel
 * set custom message for successful payment, canceled payment and failed payment
 * Ability to display Poolito payment tracking code with shortcut
 * Show payment gateway errors

== Installation ==
1. In your WordPress admin panel, go to Plugins > New Plugin
2. Find our Poolito payment gateway plugin and click Install now
3. Alternatively, download the plugin and upload the contents of poolito-wc-payment-gateway.zip to your plugins directory, which usually is /wp-content/plugins/
4. Activate the plugin.
5. Go to WooCommerce > Payment > Poolito payment gateway and set ApiKey and walletId that given before from poolito application.
6. Now enjoy!
